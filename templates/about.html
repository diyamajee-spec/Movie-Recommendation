Project Overview

About CineRec

Age-Based Multilingual Movie Recommendation System

CineRec is a state-of-the-art hybrid recommendation platform designed to demonstrate the power of **Natural Language Processing (NLP)** and **Machine Learning** in media discovery. By analyzing semantic text similarities in movie overviews and adjusting for user demographics, it delivers highly relevant, safe, and personalized suggestions.

Core Objectives

  • Semantic Discoverability: Go beyond keyword search by analyzing full movie plots using TF-IDF and Cosine Similarity models.
  • Strict Content Boundaries: Safeguard younger audiences with rule-based age filters restricting sensitive categories.
  • Multilingual Capabilities: Seamlessly serve English, Hindi, Tamil, and Telugu movie titles.
  • Premium Client UX: Provide an interactive, zero-latency filtering experience matching commercial platforms.

Recommendation Pipeline

1
Demographics Filter

First, the system extracts movies matching the chosen language and age suitability rules.

2
TF-IDF Vectorization

Analyzes plot overviews using term-frequency metrics to capture context details.

3
Cosine Similarity & Weighting

Computes the plot similarity and blends it with popularity and voting metrics for final ranking.

Age-Based Content Policies

To protect kids and teens from graphic violence, mature language, or horror, we employ strict genre filter restrictions:

Age Group Allowed Genres Safety Policy & Exclusions
Kids (Ages 5-12) Animation Family Adventure Fantasy Comedy Strictly excludes all Action, Drama, Thriller, Horror, Crime, War, Western, History, and adult-flagged movies.
Teens (Ages 13-17) All Kids genres + Action Science Fiction Drama Allows general action, drama, and sci-fi. Excludes Horror, Crime, Thriller, War, Western, History, Music, Mystery, Romance, Documentary, and adult-flagged content.
Adults (Ages 18+) All 18 genres Full catalog access with no restrictions.

Hybrid Scoring Model

Pure content text similarity can sometimes elevate obscure or low-rated titles. To deliver professional-grade suggestions, CineRec ranks movies using a weighted combination of cosine similarity, overall popularity, and user review averages:

Score = (50% * Similarity) + (30% * Popularity) + (20% * Rating)
50% Cosine Similarity

Computes the cosine angle between the query vector and the movie plot TF-IDF vector.

30% TMDB Popularity

Normalized popularity weight based on view counts and search trends across TMDB.

20% TMDB Rating

Scaled voting average representing user reviews, ensuring higher quality recommendations rise to the top.
